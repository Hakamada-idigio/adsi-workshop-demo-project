# 打刻メモ機能 — 要件 Q&A

## 確定仕様

| 項目 | 仕様 |
|------|------|
| 対象 | 出勤・退勤の両方 |
| 入力 | 任意（空でも打刻可能） |
| 文字数 | 最大100文字 |
| UI | 打刻ボタンの上に1行テキストフィールドを常時表示 |
| 表示 | 勤怠履歴画面の各日の詳細に表示 |
| 管理者閲覧 | チームメンバーのメモを閲覧可能 |
| 編集 | 打刻後も編集可能（期限なし） |
| 編集権限 | 本人 + 管理者 |
| 保存形式 | 出勤メモ・退勤メモを別カラムで保存 |
| 用途 | イレギュラーな勤怠時の備忘（自由記述） |
| レポート | 月次レポート（PDF）に備考列として含める |

---

## 基本仕様

[Question] メモは出勤・退勤の両方に入力できるようにしますか？それとも片方のみ？
[Answer]
両方にお願いします
[Question] メモの入力は任意（空でも打刻可能）ですか？それとも必須にしますか？
[Answer]
任意入力
[Question] メモの最大文字数に制限はありますか？（例: 100文字、200文字、制限なし）
[Answer]
100文字
## UI / UX

[Question] メモ入力欄はどこに表示しますか？
